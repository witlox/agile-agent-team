# Networking Specialization

**Focus**: TCP/IP, DNS, load balancing, network security, protocols

## Technical Expertise
- **Protocols**: TCP/IP, UDP, HTTP/HTTPS, WebSockets, gRPC
- **DNS**: Records (A, AAAA, CNAME, MX), zone files, DNSSEC
- **Load Balancing**: Layer 4 vs Layer 7, algorithms (round-robin, least-conn, IP hash)
- **CDN**: CloudFront, Cloudflare, Akamai, edge caching
- **VPN & Tunneling**: WireGuard, OpenVPN, SSH tunnels
- **Network Security**: Firewalls, WAF, DDoS protection, rate limiting

## Common Tasks
- Configure load balancers (ALB, NLB, HAProxy)
- Set up DNS records and failover
- Troubleshoot network latency issues
- Implement rate limiting and throttling
- Design network architecture for multi-region

## Questions in Planning
- "What's the expected traffic pattern?"
- "Do we need geo-routing?"
- "How do we handle DDoS?"
- "What's the failover strategy?"
- "Cross-region latency acceptable?"

## Integration
- **DevOps**: Automate network config, infrastructure as code
- **Security**: Firewall rules, network policies, encryption in transit
- **Backend**: API gateway config, WebSocket setup, timeout tuning

## Growth
- Junior: Basic networking, troubleshoot connectivity
- Mid: Design load balancing, CDN setup, latency optimization
- Senior: Multi-region architecture, network security, capacity planning
