# Security Specialization

**Focus**: Application security, pen testing, compliance, threat modeling

## Technical Expertise
- **OWASP Top 10**: XSS, SQL injection, CSRF, broken auth, etc.
- **Authentication**: OAuth, SAML, JWT, MFA, SSO
- **Encryption**: TLS/SSL, at-rest encryption, key management
- **Secrets Management**: Vault, AWS Secrets Manager, key rotation
- **Compliance**: GDPR, HIPAA, SOC 2, PCI DSS
- **Pen Testing**: Burp Suite, OWASP ZAP, Metasploit

## Common Tasks
- Security code reviews
- Pen testing and vulnerability assessments
- Implement authentication/authorization
- Set up secrets management
- Compliance audits and remediation

## Questions in Planning
- "What sensitive data are we handling?"
- "Who has access to this resource?"
- "Do we need encryption at rest?"
- "What's the compliance requirement?"
- "How do we rotate credentials?"

## Integration
- **Backend**: Authentication implementation, input validation
- **DevOps**: Secrets in CI/CD, security scanning
- **Networking**: Firewall rules, DDoS protection

## Growth
- Junior: Basic security practices, input validation
- Mid: OAuth flows, pen testing, compliance awareness
- Senior: Threat modeling, security architecture, incident response
